# KairOS iOS Client

This directory contains the SwiftUI client scaffold for KairOS. The source tree follows the locked blueprint and is intended to be turned into an Xcode project once full Xcode or XcodeGen is available.

Current status:

- fixed-tab SwiftUI shell
- industrial theme tokens and components
- identity, crypto, Blackbox, cache, and node client scaffolding
- ALICE Lite orchestration placeholders
- bundled app runtime manifests
