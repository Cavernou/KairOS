KAIROS NODE HELP

                
            

            

                

                    
DASHBOARD

                    
DEVICES: View connected KairOS devices and their status.

                    
ADMIN CODE: Registration code for activating new devices. Click REFRESH CODE to generate a new code.

                    
SYSTEM STATUS: Overview of system statistics including devices, contacts, messages, files, and admin codes.

                    
NETWORK: Network configuration including tailnet name and listening ports.

                    
CLOCK: System clock display with 12-hour, 24-hour, and UTC modes.

                


                

                    
DEVICES

                    
DEVICES: View and manage connected KairOS devices.

                    
CONTACTS: Manage contact list with K-numbers and display names.

                


                

                    
COMMUNICATIONS

                    
CALLING: Initiate and manage voice calls to other KairOS devices.

                    
FILTERING: Set up message filtering rules to block or allow content based on keywords.

                


                

                    
FILES

                    
FILE TRANSFER: Send files to connected KairOS devices.

                    
MEDIA: Upload and manage image, audio, and video files.

                    
STORAGE: View disk usage statistics. CLEANUP OLD FILES deletes telemetry events older than 30 days and media files older than 90 days to free up storage space.

                    
NOTES: Create, view, and delete text notes.

                    
CALENDAR: Create, view, and delete calendar events.

                    
TASKS: Create, view, and delete tasks with priorities and due dates.

                    
TELEMETRY: View system activity logs and export them.

                


                

                    
SETTINGS

                    
LISTEN ADDR: Network address and port for gRPC server (e.g., 0.0.0.0:8080)

                    
MOCK HTTP ADDR: Network address and port for HTTP gateway (e.g., 0.0.0.0:8081)

                    
TAILNET: Tailscale tailnet name for your network (e.g., kairos.ts.net)

                    
TAILSCALE ENABLED: Enable Tailscale VPN for secure networking

                    
QUEUE RETRY LIMIT: Maximum number of retry attempts for failed messages

                    
QUEUE TTL (HOURS): Time-to-live for queued messages before expiring

                    
ADMIN CODE INTERVAL (SEC): How long admin codes remain valid before rotating

                    
SOUNDS: Browse and play available sound effects organized by category.
