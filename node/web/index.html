KAIROS NODE HELP

                
            

            

                

                    
DASHBOARD

                    
DEVICES: View connected KairOS devices and their status. Shows device name, K-number, connection status, and last activity timestamp. Devices are automatically detected when they connect to the tailnet.

                    
ADMIN CODE: Registration code for activating new devices. Each code is valid for a limited time (default: 1 hour). Click REFRESH CODE to generate a new code. New devices must enter this code during activation to join the network.

                    
SYSTEM STATUS: Overview of system statistics including total devices, total contacts, pending messages, queued files, and active admin codes. Updates automatically every 5 seconds.

                    
NETWORK: Network configuration including tailnet name, listening address, Tailscale connection status, and reachability. Shows whether Tailscale is enabled and connected.

                    
CLOCK: System clock display with 12-hour, 24-hour, and UTC modes. Click on the clock to toggle between modes. Shows current time and date.

                    
LIVE STATISTICS: Real-time system metrics including CPU usage, memory usage, goroutine count, and uptime. Updates automatically every 5 seconds using Go runtime statistics.

                    
NEWS BANNER: Displays broadcast messages from the node administrator. Shows message content, priority (high/medium/low), and expiration time. Click X to dismiss the banner.

                


                

                    
DEVICES

                    
DEVICES: View and manage connected KairOS devices. Each device shows K-number, display name, status (online/offline), and last interaction time. Devices can be deleted from the network by clicking DELETE.

                    
CONTACTS: Manage contact list with K-numbers, display names, real phone numbers, notes, trust status (trusted/pending/unknown), and last interaction time. Contacts can be added, viewed, and deleted. Trust status indicates whether the contact is verified.

                


                

                    
COMMUNICATIONS

                    
CALLING: Initiate and manage voice calls to other KairOS devices. Select a contact from the dropdown and click CALL to initiate. Active calls show duration and can be ended by clicking END CALL.

                    
FILTERING: Set up message filtering rules to block or allow content based on keywords. Add filters by entering keywords and selecting action (block/allow). Filters are applied to incoming messages. Click DELETE to remove a filter.

                    
MESSAGES: View message queue status. Shows queued messages, delivery attempts, and retry status. Messages are automatically retried with exponential backoff if delivery fails.

                


                

                    
BROADCAST

                    
BROADCAST NEWS: Send broadcast messages to all connected devices. Enter message text, select priority (high/medium/low), and set expiration time (1 hour to 7 days). Broadcasts appear as a banner on the dashboard and are stored on the node.

                    
ACTIVE BROADCASTS: View currently active broadcast messages. Shows message content, priority level, expiration date/time, and status (active/expired). Only one broadcast is active at a time.

                


                

                    
FILES

                    
FILE BROWSER: Navigate the node's file system. Use breadcrumbs to navigate to parent directories. Click on directories to enter them. Click on files to view their contents. Use the search box to filter files by name. Click REFRESH to reload the current directory. Click ROOT to go to the top-level directory. Click UP to go to the parent directory.

                    
FILE VIEWER: View the contents of text files. Shows file path and content in a scrollable viewer. Click CLOSE to return to the file browser.

                    
FILE TRANSFER: Send files to connected KairOS devices. Select a file from your computer, choose a target device from the dropdown, and click SEND FILE to initiate transfer.

                    
MEDIA: Upload and manage image, audio, and video files. Shows file name, type, and size. Click DELETE to remove media files. Media files are stored in the node's media directory.

                    
STORAGE: View disk usage statistics including total capacity, used space, free space, and breakdown by category (files, messages, media). Click CLEANUP OLD FILES to delete telemetry events older than 30 days and media files older than 90 days to free up storage space.

                    
NOTES: Create, view, and delete text notes. Enter title and content, then click ADD NOTE to save. Notes are stored in the database and can be deleted by clicking DELETE.

                    
CALENDAR: Create, view, and delete calendar events. Enter title, description, start time, end time, location, and attendees. Click ADD EVENT to save. Events can be deleted by clicking DELETE.

                    
TASKS: Create, view, and delete tasks with priorities and due dates. Enter title, description, due date, and priority (high/medium/low). Click ADD TASK to save. Tasks can be deleted by clicking DELETE. Tasks are sorted by priority and due date.

                    
TELEMETRY: View system activity logs including event type, details, device ID, and timestamp. Click EXPORT LOG to download the full telemetry log as a JSON file.

                


                

                    
SETTINGS

                    
LISTEN ADDR: Network address and port for gRPC server (e.g., 0.0.0.0:8080). This is the address where the node listens for gRPC connections from other KairOS devices. Use 0.0.0.0 to listen on all network interfaces.

                    
MOCK HTTP ADDR: Network address and port for HTTP gateway (e.g., 0.0.0.0:8081). This is the address where the web UI is accessible. Use 0.0.0.0 to listen on all network interfaces.

                    
TAILNET: Tailscale tailnet name for your network (e.g., kairos.ts.net). This must match your Tailscale network name. Devices on the same tailnet can communicate securely.

                    
TAILSCALE ENABLED: Enable Tailscale VPN for secure networking. When enabled, only devices from the configured tailnet can access the node. Disable for testing or local-only operation.

                    
QUEUE RETRY LIMIT: Maximum number of retry attempts for failed messages. Messages that fail to deliver will be retried this many times with exponential backoff before being marked as failed.

                    
QUEUE TTL (HOURS): Time-to-live for queued messages before expiring. Messages older than this time will be automatically deleted from the queue even if not delivered.

                    
ADMIN CODE INTERVAL (SEC): How long admin codes remain valid before rotating. New codes are generated automatically after this interval expires. Default is 3600 seconds (1 hour).

                    
SOUNDS: Browse and play available sound effects organized by category. Sounds include click sounds, alerts, success sounds, ambient sounds, UI sounds, notifications, and other effects. Click PLAY to preview a sound. Sounds are used throughout the interface for feedback.
